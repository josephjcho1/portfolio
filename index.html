Education  ·  RAG Assistant

UniMatch

Your personalized, private admissions counselor.
[ Ollama + Jan Nano + LangChain + ChromaDB ]
Problem
Many students cannot access personalized university admissions advice without paying expensive consulting fees or sacrificing their personal data to online AI platforms..
Process
  • Base model: Jan Nano (locally through Ollama) for its low parameter count and strong performance in agentic workflows (e.g. web search and RAG).
  • Web search: Integrated DuckDuckGo Search to prioritize user privacy and avoid unnecessary data collection.
  • RAG framework: Built the retrieval pipeline using LangChain and ChromaDB for efficient document retrieval.
  • Testing environment: Developed and tested the system in Google Colab for rapid iteration and experimentation.
Outcome
Presented the program at a school assembly, piloted with 20 students, with ~80% continued using throughout admissions seasons.
Reflection
Building UniMatch made me more aware that many social problems, like inequality in education through unequal access to valuable information, evolve over time and often require modern, technology-driven solutions to meaningfully address them.

KSEAU Website

Transforming an offline scientific community into a structured, credible digital platform.
Information Architecture CMS [ HTML + CSS + SQL ]
Problem
The newly formed organization The Korean Scientists and Engineers Association in the UAE (KSEAU) lacked a professional, centralized digital presence, making it difficult to present the organization credibly to external institutions and meet formal requirements for recognition under The Korean Federation of Science and Technology Societies (KOFST).
Process
  • Consulted with board members and institutional requirements of the website's contents and direction.
  • Built and launched the association's official website end-to-end using Wordpress Elementor + Domain.com hosting
  • Created documentation and conducted a handoff session to train a coordinator for independent site management after project completion.
Outcome
Successfully delivered a fully functional official website for KSEAU, directly contributing to the association’s recognition as an official associate member under KOFST and earning the recognition as the association’s first-ever junior member for delivering both technical and organizational impact.
Reflection
This experience showed me how technical development can directly influence institutional credibility and organizational opportunities, especially when combined with clear documentation and long-term usability planning.

BookShelf

Encouraging and incentivizing healthy reading habits.
[ tauri + next.js + Replit ]
Problem
My reading habit was scattered and all over the place, as I struggled to keep track of what I've already read and what I want to read.
Process
  • Design: sketched out my vision for the simple application on Procreate.
  • Rapid AI-assisted development workflow: built the core application logic and interface using Replit + Claude code to accelerate iteration and feature design.

Underwater Robotics  ·  2025 MATE ROV Competition

MATE ROV

Designing, building, and competing a cost effective and lightweight ROV.
[ Raspberry Pi + ArduSub + QGroundControl ]
Problem
Build a tethered ROV to compete in the 2025 MATE ROV competition hosted in Alexandria, Egypt, within the strict size (<5.5kg) and budget limits (2000AED).
Process
  • Designed and built a lightweight ROV using an acrylic frame reinforced with stainless steel structure for easy disassembly-assembly and strong durability.
  • Engineered a sealed electronics system with gasketed watertight enclosure, epoxy cable penetrations, and regulated 12V power distribution.
  • Developed a 6-thruster propulsion system with Pixhawk + Raspberry Pi control architecture using ArduSub, MAVLink, and QGroundControl.
  • Calibrated buoyancy using calculated ballast to achieve near-neutral stability for precise underwater maneuvering.
  • Validated system through iterative testing (waterproofing, mobility, and full-system diagnostics) while prioritizing cost efficiency and reliability.
Outcome
Compiled progress into a coherent 20-page technical report. Although the ROV encountered failure during a deep water retrieval mission, we were awared the 'Team Spirit Award' in recognition of strong collaboration, resilience, and the quality of the technical documentation.
Reflection
This project taught me how important iteration, testing, and failure analysis are in engineering, and how strong teamwork can still lead to meaningful success even when the final system does not perform as intended.
